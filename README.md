## 🚀 Deploying to GitHub Pages (Free Hosting)

```bash
# Step 1 — Initialize git (if not done already)
git init

# Step 2 — Add all files
git add .

# Step 3 — Commit
git commit -m "Initial portfolio commit"

# Step 4 — Connect to GitHub repo
git remote add origin https://github.com/Ravi12-max/portfolio.git

# Step 5 — Push
git push -u origin main
```

Then go to your repo on GitHub:
**Settings → Pages → Source → Deploy from branch → main → Save**

Your live URL will be: `https://ravi12-max.github.io/portfolio`
